## Getting Started

Let's explore each component with real examples.

<Steps>

### Installation

First, ensure all components are imported in your MDX file:

```typescript
import {
  Alert,
  Badge,
  Card,
  Steps,
  Tabs,
  Accordion,
  Quote,
  Timeline,
  Kbd,
} from "@/components/mdx";
```

### Using Components

Simply use them in your MDX content like regular React components:

```mdx
<Alert variant="success">Your component is ready to use!</Alert>
```

### Customization

All components support theme-aware styling and can be customized via props.

</Steps>

## Code Examples

### Regular Code Block

```typescript
interface User {
  id: string;
  name: string;
  email: string;
}

async function getUser(id: string): Promise<User> {
  const response = await fetch(`/api/users/${id}`);
  return response.json();
}
```

### Code Comparison

<CodeComparison
  language="typescript"
  beforeLabel="❌ Don't do this"
  afterLabel="✅ Do this instead"
  before={`// Nested callbacks
fetchUser((user) => {
  fetchPosts(user.id, (posts) => {
    fetchComments(posts[0].id, (comments) => {
      console.log(comments);
    });
  });
});`}
  after={`// Async/await
const user = await fetchUser();

console.log(comments);`}
