# Бинар хайлтын алгоритм (Binary Search)

Бинар хайлт нь эрэмбэлэгдсэн жагсаалтаас тухайн элементийг олоход ашиглагддаг хамгийн үр дүнтэй аргуудын нэг юм. 



## Энэ нь хэрхэн ажилладаг вэ?

Бинар хайлт нь "хуваагаад засаглах" (divide and conquer) зарчмаар ажилладаг. Алхам бүрт хайх мужийг хоёр дахин багасгадаг:

1.  Жагсаалтын дундах элементийг олж хайна.
2.  Хэрэв дундах элемент нь бидний хайж буй утгатай тэнцүү бол хайлт дуусна.
3.  Хэрэв хайж буй утга нь дундах элементээс **бага** бол жагсаалтын зүүн талд хайлтыг үргэлжлүүлнэ.
4.  Хэрэв хайж буй утга нь дундах элементээс **их** бол жагсаалтын баруун талд хайлтыг үргэлжлүүлнэ.
5.  Энэ процессыг утга олдох хүртэл эсвэл хайх муж үгүй болох хүртэл давтана.



## Хугацааны нарийн төвөгтэй байдал

Бинар хайлт нь маш хурдан бөгөөд өгөгдлийн хэмжээ ихсэх тусам түүний давуу тал илүү тод харагддаг.

* **Best Case:** $O(1)$ (Хайж буй утга яг голд нь байх үед)
* **Average / Worst Case:** $O(\log n)$

## TypeScript дээрх жишээ код

Дараах код нь итератив (iterative) хэлбэрээр бичигдсэн бинар хайлтын жишээ юм:

```typescript
function binarySearch(arr: number[], target: number): number {
  let left = 0;
  let right = arr.length - 1;

  while (left <= right) {
    const mid = Math.floor(left + (right - left) / 2);

    if (arr[mid] === target) {
      return mid; // Утга олдсон индекс
    }

    if (arr[mid] < target) {
      left = mid + 1; // Баруун талыг хайх
    } else {
      right = mid - 1; // Зүүн талыг хайх
    }
  }

  return -1; // Утга олдоогүй
}

const numbers = [2, 5, 8, 12, 16, 23, 38, 56, 72, 91];
console.log(binarySearch(numbers, 23)); // Гаралт: 5
